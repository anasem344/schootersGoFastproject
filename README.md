# scootersGo

## Описание проекта

Аналитический проект по исследованию данных сервиса аренды электросамокатов GoFast.
Цель анализа — изучить поведение пользователей, сравнить пользователей с подпиской и без неё, а также оценить влияние подписки на использование сервиса и выручку.

## Используемые данные

В проекте используются три набора данных:

**users_go.csv**

* `user_id` — уникальный идентификатор пользователя
* `name` — имя пользователя
* `age` — возраст
* `city` — город
* `subscription_type` — тип подписки (free / ultra)

**rides_go.csv**

* `user_id` — идентификатор пользователя
* `distance` — расстояние поездки (м)
* `duration` — длительность поездки (мин)
* `date` — дата поездки

**subscriptions_go.csv**

* `subscription_type` — тип подписки
* `minute_price` — стоимость минуты
* `start_ride_price` — стоимость старта поездки
* `subscription_fee` — ежемесячная плата

## Задачи анализа

В рамках проекта были выполнены следующие этапы:

* загрузка и первичная проверка данных
* обработка пропусков и удаление дубликатов
* исследовательский анализ данных (EDA)
* анализ пользователей по городам и возрасту
* анализ длительности и расстояния поездок
* объединение таблиц и подготовка данных
* расчёт месячной выручки пользователей
* проверка статистических гипотез

## Основные выводы

* В сервисе зарегистрировано 1565 пользователей и 18 068 поездок.
* Большинство пользователей используют сервис без подписки, однако доля пользователей с подпиской также значительна.
* Средняя длительность поездки составляет около 18 минут.
* Пользователи с подпиской проводят в поездках больше времени, чем пользователи без подписки.
* Статистический тест показал значимое различие в длительности поездок между группами.
* Нельзя утверждать, что средняя дистанция поездки пользователей с подпиской превышает оптимальное значение 3130 метров.
